# Job Application Tracker

I decided to apply to 50 job applications every day until I land a great role.

This repository tracks my progress, mindset, learning, and improvements in my job search journey.

## What I Track
- Number of jobs applied
- Responses & interviews
- Rejections
- Skills I improve
- Lessons I learn

## Why I'm Doing This
I believe in consistency, focus, and self-improvement. Instead of waiting for luck, I'm taking action daily.

## Tools I Use
- Google Sheets
- LinkedIn & Indeed
- GitHub

## Goal
To build a meaningful career and grow every single day.
